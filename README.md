# Bloc de Notas (Python + Tkinter)
Aplicación de escritorio simple hecha en Python con Tkinter para crear, abrir y guardar notas de texto.
## Características
- Interfaz gráfica con área de texto.
- Menú **Archivo** con opciones:
  - Abrir archivo `.txt`
  - Guardar archivo `.txt`
  - Salir de la aplicación
- Manejo de errores con mensajes en pantalla.
## Requisitos
- Python 3.10+ (recomendado)
- Tkinter (incluido normalmente en instalaciones estándar de Python)
